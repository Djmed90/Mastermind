Voice Assistant
Overview
This project is a voice assistant application that utilizes natural language processing (NLP) and python librarys to create a seemless voice assistant that will answer many questions utalizing openAI's ChatGPT language model.

Features:
1 . Transcribes spoken words into text using the Google Speech Recognition API.
2 . Processes user queries and generates contextually relevant responses using OpenAI's GPT-3 model.
3 . Supports seamless integration with various audio input and output devices.
